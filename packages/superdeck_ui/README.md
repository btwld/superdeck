# SuperDeck UI

A UI components library for SuperDeck with mix and remix support.

## Features

- Built with [Mix](https://pub.dev/packages/mix) for styling
- Powered by [Remix](https://pub.dev/packages/remix) for state management
- Seamless integration with SuperDeck presentation framework

## Installation

Add this package to your `pubspec.yaml`:

```yaml
dependencies:
  superdeck_ui: ^0.0.1
```

## Development

This package is part of the SuperDeck workspace. To set up for development:

```bash
# Ensure you're using the stable Flutter version
fvm use stable --force

# Install dependencies
melos bootstrap

# Run analysis
melos run analyze

# Run tests
melos run test
```
